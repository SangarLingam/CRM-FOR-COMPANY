# DesignCRM – Interior Project Management CRM

DesignCRM is a **Customer Relationship Management (CRM) system** built for interior design businesses to manage customers, projects, quotes, and payments in one place.

The application is built with **Next.js, MongoDB, and deployed on Vercel**.

---

## 🚀 Features

- Customer management
- Project tracking
- Quote generation
- Payment tracking
- Role based access
  - Super Admin
  - Manager
  - Sales
  - Designer
- Dashboard analytics
- Secure authentication
- Responsive modern UI

---

## 🛠 Tech Stack

**Frontend**
- Next.js (App Router)
- React
- Tailwind CSS

**Backend**
- Next.js API Routes
- Node.js

**Database**
- MongoDB

**Deployment**
- Vercel
